# YAML
Yaml is a YAML parser for yaml version 1.2.
Yaml 1.2 syntax is specified in the official specification release at https://yaml.org/spec

## Backward Compatibility
This parser will start out to be fully compliant with the revision 1.2.x 
dated 2021-10-01 on https://yaml.org/spec/1.2.2/  
Compatibility with specifications 1.1.x will come in future releases.
